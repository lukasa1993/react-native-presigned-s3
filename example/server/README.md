# sample s3-signer


### Setup
1. this is for NY DigitalOcean [Spaces](https://www.digitalocean.com/products/spaces) which is S3 compatible
2. Setup env
    ```dotenv
    SPACES_KEY={{s3_key}}
    SPACES_SECRET={{s3_secret}}
    BUCKET={{s3_bucket}}
    ```
3. Run yarn and dev to jump start
   1. ```yarn```
   2. ```yarn dev```
